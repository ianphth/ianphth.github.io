Connecty

An user interface prototype for social application.

Outlines: Home , Chat , Connections , Messages , Friends

Home

While the application has to provide information for time and space of every users, the principle is to show the real-time information to the user on home page. Users can gain access to detailed recent or closest friends' information by clicking on big, obvious icons. Also, they can get to full friend list by clicking on small, simple icons.

Certainly, we need to see the most recent location of a certain friend and show the location to the user. Swiping enables users to see other nearest friends' location.

Also, it acts as a normal chatroom. So users will have the need to see the newest messages, too.

Since the look-up-info icon in the upper left corner is not often used, upper-left is a decent place to avoid users touching the icon.

Chat

The focus of chat room is to put user into opponent's shoes. So the basic element is to have relevant sight. All other elements have to meet this principle.

The background will reveal the picture of opponent's location, which is searched on internet.

Text is comprised of basic color. No colored edge or shadow are featured in order to accompany with background picture.

Connections

Connections page shows map to trace location of users' friends. They will be tagged on the map.

Messages

Messages page shows all the messages users have gotten. It's arranged by chronological order.

Upper right icon provides a function for starting new chat.

Friends

Friends page shows the last known location of users' friends. To maintain its convenience, search bar is presented on the bottom of the screen. Also, the arrangement can be altered by time, name or location.
