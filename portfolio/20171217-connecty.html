Connecty

An user interface prototype for social application.

Outlines: Home , Messages , Connections

Home

While the application has to provide information for time and space of every users, the principle is to show the real-time information to the user on home page.

Certainly, we need to see the most recent location of a certain friend and show the location to the user. Swiping enables users to see other nearest friends' location.

Also, it acts as a normal chatroom. So users will have the need to see the newest messages, too.

Messages

The focus of message page is to put user into opponent's shoes. So the basic element is to have relevant sight. All other elements have to meet this principle.

The background will reveal the picture of opponent's location, which is searched on internet.

Text is comprised of basic color. No colored edge or shadow are featured in order to accompany with background picture.
